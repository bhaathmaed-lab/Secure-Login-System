# Secure Login System 🔐

A secure user authentication system built using **Python Flask** and **SQLite**.  
This project demonstrates secure login practices including password hashing, database security, and session authentication.

## 📌 Project Overview

The Secure Login System allows users to:
- Register a new account
- Login securely
- Store passwords using encryption hashing
- Manage user authentication using sessions

## 🚀 Features

✅ User Registration  
✅ Secure Login Authentication  
✅ Password Hashing using Bcrypt  
✅ SQLite Database Integration  
✅ SQL Injection Prevention  
✅ User Session Management  
✅ Input Validation  
✅ Secure Password Storage  

## 🛠️ Technologies Used

- Python
- Flask Framework
- SQLite Database
- HTML
- CSS
- Flask-Bcrypt
