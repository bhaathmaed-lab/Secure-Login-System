# Secure Login Authentication System

A secure web-based login and registration system developed using HTML, CSS, PHP, and MySQL. The project implements cybersecurity best practices such as password hashing, session management, and SQL injection prevention through prepared statements.

## Features

* User Registration
* User Login Authentication
* Password Hashing (bcrypt)
* Session Management
* Secure Logout
* SQL Injection Protection
* Responsive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL

## Security Features

* Prepared Statements
* Password Hashing
* Session-Based Access Control
* Input Validation

## Future Enhancements

* Two-Factor Authentication (2FA)
* Email Verification
* Password Reset Functionality
* Login Attempt Limiting
* CAPTCHA Integration
